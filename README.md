# shell-todo
Simple ToDo for current directory.

# Example
```
$ cd work1
$ todo eat SPAM!
     1  eat SPAM!
$ cd ..

few days later...

$ cd work1
     1  eat SPAM!
```

# Install

for bash or zsh:
```bash
source shell-todo
```

use antigen:
```bash
antigen temmings/shell-todo shell-todo
```
